# python-twitter-menfess-bot

Bot untuk akun twitter autobase menfess. Semua pesan yang dikirim ke akun dan mengandung kata kunci akan menjadi tweet

## install

`pip install -r requirements.txt`

## config

1. buka `config.py`
2. edit semua key dan token

## run

`python bot.py`
